# Epidemic-ststistics-clustering
Кластеризация регионов РФ по эпидемиологическим показателям с использованием методов снижения размерности (UMAP) и кластеризации (HDBSCAN). Визуализация результатов и анализ схожести регионов.


# Кластеризация регионов РФ по эпидемиологическим показателям

Этот репозиторий содержит код для анализа эпидемиологической статистики по регионам Российской Федерации. Основная цель проекта — выявить группы регионов со схожей динамикой эпидемиологических показателей с помощью современных методов машинного обучения без учителя.

## Используемые методы

Для решения задачи кластеризации высокоразмерных данных используется двухэтапный подход:
1. **Снижение размерности (UMAP):** Алгоритм UMAP (Uniform Manifold Approximation and Projection) используется для снижения размерности данных с сохранением локальной и глобальной структуры данных. Это позволяет визуализировать многомерные данные в 2D или 3D пространстве.
2. **Кластеризация (HDBSCAN):** Алгоритм HDBSCAN (Hierarchical Density-Based Spatial Clustering of Applications with Noise) используется для кластеризации данных в пространстве пониженной размерности. Этот метод устойчив к шуму и не требует предварительного задания количества кластеров.

# Как использовать

1. Клонируйте репозиторий
```bash
git clone https://github.com/your-username/epidemic-statistics-clustering.git
cd epidemic-statistics-clustering
```

2. Установите необходимые библиотеки.
3. Откройте файл Epidem_statistic_Iers_Pseud.ipynb в Jupyter Notebook или JupyterLab.
4. Загрузите ваши данные (убедитесь, что путь к файлу данных в ноутбуке указан верно).
5. Запустите ячейки ноутбука последовательно.
